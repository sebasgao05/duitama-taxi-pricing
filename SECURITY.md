## Reportar una vulnerabilidad

**No abras un issue público para reportar vulnerabilidades de seguridad.**

Envía un reporte privado usando [GitHub Security Advisories](../../security/advisories/new) con:

- Descripción del problema
- Pasos para reproducirlo
- Impacto potencial

Recibirás respuesta en un plazo de **5 días hábiles**. Si se confirma la vulnerabilidad, se publicará un advisory y se te dará crédito en el fix.
